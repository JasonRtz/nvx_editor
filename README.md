## Overview
NVK Editor is a simple text editor designed for speed and ease of use. Leveraging the **Qt application framework**, it provides a responsive interface and essential features for everyday use.

## Features
* **Lightweight Performance:** Fast startup times and low memory footprint.
* **Cross-Platform:** Runs on Windows and Linux thanks to Python and Qt.
* **Clean UI:** Minimalist design to help you focus on your content.

## Dependencies
Before running the editor, ensure you have **Python 3.x** installed. You will also need the following library:

* **PyQt6**
  
To install the required dependency, run:
```bash
pip install PyQt6
```
## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/JasonRtz/nvx_editor.git
   cd nvx_editor

2. **Run the app:**
   ```bash
   python ./main.py
   ```
